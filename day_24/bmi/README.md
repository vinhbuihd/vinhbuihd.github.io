## Ứng dụng tính chỉ số BMI

Công thức tính:

```
BMI = weight / (height * height)

Trong đó:
- weight : Cân nặng (kg)
- height : Chiều cao (m)
```

Bao gồm các chức năng:
- Check validate ở các ô
- Tính toán chỉ số và hiển thị
